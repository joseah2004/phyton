## ¿Que he realizado?

En este proyecto he trabajado con Python a un nivel básico, centrándome en comprender su sintaxis sencilla y cómo se estructura un programa. 
He aprendido a usar variables, tipos de datos y condicionales para resolver pequeños problemas. 
También he practicado con funciones, entendiendo cómo se crean, cómo reciben parámetros y cómo devuelven resultados. 
Estos ejercicios me han ayudado a mejorar mi lógica y a escribir código más ordenado.
